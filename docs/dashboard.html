D2 — concurrency não-dedup
concurrency.group em ci.yml não deduplica runs push:claude/** vs pull_request:synchronize: github.ref diverge entre os dois eventos (refs/heads/... vs refs/pull/N/merge). Resultado: dois runs verdes redundantes por push enquanto PR aberta. Candidato a concurrency.group: ${{ github.workflow }}-${{ github.event.pull_request.number || github.ref }} em sprint futura. Aberta.
